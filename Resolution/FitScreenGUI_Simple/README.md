Landscape - FIXED_HEIGHT 高度基准

换算公式:

设计分辨率.h = 标准分辨率.h
缩放比 = 屏幕分辨率.h / 标准分辨率.h
设计分辨率.w = 屏幕分辨率.w / 缩放比
无效边距 = (设计分辨率.w - 标准分辨率.w) / 2
